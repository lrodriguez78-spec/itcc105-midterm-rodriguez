# ASSCAT Campus AI Copilot

## Problem Statement
The old student portal is confusing and hard to use because it has too many menus and is not user-friendly.

## Industry 5.0 Justification
This system uses AI to assist students, not replace humans. It helps make tasks faster and easier.

## User Stories
1. As a student, I want to check my clearance using AI so that I don’t need to navigate many menus.
2. As a student, I want to view my grades quickly so that I can track my performance.
3. As a student, I want to enroll subjects using AI so that the process becomes easier.
